If you're seeing 404 errors for static assets, follow these steps:

Step 1: Clear Browser Cache

Chrome/Edge: Press Ctrl+Shift+Delete, select "Cached images and files", click "Clear data"

Firefox: Press Ctrl+Shift+Delete, select "Cache", click "Clear Now"

Step 2: Hard Refresh

Press Ctrl+F5 or Ctrl+Shift+R to force reload the page

Step 3: Check Server Port

Make sure you're accessing the correct port shown in your terminal:

  • If terminal shows http://localhost:3000 - use that
  • If terminal shows http://localhost:3001 - use that
  • If terminal shows http://localhost:3002 - use that

Step 4: Restart Development Server

In your terminal, stop the server (Ctrl+C) and run:

npm run dev
